# BAPS - BACS Online Programming Camp 2020

My notes and solutions while following the free [BAPS - BACS Online Programming Camp, 2020](https://www.youtube.com/playlist?list=PLWtSipmftM8qbtfj2WkoQ7Z9ZrhCl6rPz) playlist on YouTube.

Organized by [BAPS](https://baps-bgd.github.io/camp/) (Bangladesh Association of Problem Setters) in association with BACS.

Folders are numbered by playlist order (starting at `00`). Topics with multiple sessions use a parent folder (numbered by the first session) and numbered subfolders for each video.
